## 介紹

精簡版是基於 [vue-pure-admin](https://github.com/pure-admin/vue-pure-admin) 提煉出的骨架，包含主要功能，更適合實際專案開發，打包後的大小在全域引入 [element-plus](https://element-plus.org) 的情況下仍然低於 `2.3MB`，並且會永久同步完整版的程式碼。開啟 `brotli` 壓縮和 `cdn` 替換本地函式庫模式後，打包大小低於 `350kb`

## 版本選擇

目前為[國際化版本](https://github.com/pure-admin/pure-admin-thin/tree/i18n)

```bash
git checkout -- .vscode/settings.json
git checkout -b i18n origin/i18n
```
